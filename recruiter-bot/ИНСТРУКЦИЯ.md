## 📁 Структура проекта

```
recruiter-bot/
├── app.py              # Основное приложение (Streamlit + Gemini)
├── requirements.txt    # Зависимости Python
├── README.md          # Документация
```

## 🎯 Как запустить локально (для тестирования)

```bash
cd recruiter-bot
pip install -r requirements.txt
streamlit run app.py
```

Приложение откроется по адресу: http://localhost:8501

## 🌐 Как разместить на GitHub Pages

**Важно:** Streamlit нельзя разместить напрямую на GitHub Pages, так как это статический хостинг.

### Вариант 1: Hugging Face Spaces (БЕСПЛАТНО, рекомендуется)

1. Зарегистрируйтесь на https://huggingface.co/
2. Нажмите "New Space"
3. Выберите:
   - Space type: **Space**
   - License: **MIT**
   - SDK: **Streamlit**
4. Загрузите файлы из папки `recruiter-bot`:
   - `app.py`
   - `requirements.txt`
   - `README.md`
5. Через 2-3 минуты приложение будет доступно по ссылке вида:
   `https://huggingface.co/spaces/ВАШ_НИК/recruiter-bot`

### Вариант 2: Render.com (БЕСПЛАТНО)

1. Зарегистрируйтесь на https://render.com/
2. Создайте новый **Web Service**
3. Подключите ваш GitHub репозиторий
4. Настройки:
   - Build Command: `pip install -r requirements.txt`
   - Start Command: `streamlit run app.py --server.port $PORT --server.address 0.0.0.0`
5. Приложение будет доступно по ссылке вида:
   `https://recruiter-bot.onrender.com`

### Вариант 3: Railway.app (БЕСПЛАТНО с ограничениями)

1. Зарегистрируйтесь на https://railway.app/
2. Создайте новый проект → Deploy from GitHub repo
3. Добавьте переменную окружения (не требуется, ключ уже в коде)
4. Railway автоматически определит Streamlit и запустит приложение

## 🔧 Как добавить ваши проекты из Google Таблицы

Сейчас в коде есть 2 тестовых проекта. Чтобы загрузить ваши реальные данные:

### Шаг 1: Откройте файл `app.py`

Найдите функцию `load_projects_data()` (строка ~17)

### Шаг 2: Добавьте ваши проекты

Пример формата:

```python
projects = {
    "впрок екатеринбург": {
        "название": "Впрок Екатеринбург",
        "город": "Екатеринбург",
        "зарплата": "45000-55000 руб.",
        "график": "5/2, 8:00-17:00",
        "тип_занятости": "Полная занятость",
        "преимущества": ["Официальное трудоустройство", "Обучение", "Карьерный рост"],
        "адрес": "г. Екатеринбург, ул. Примерная, 123"
    },
    # Добавьте остальные 18 проектов здесь
}
```

### Шаг 3: Обновите базу знаний

Найдите функцию `load_objection_knowledge()` (строка ~50) и добавьте ваши скрипты из Google Doc.

## 💡 Пример использования

1. Рекрутер открывает веб-интерфейс
2. Вводит: **"Впрок Екатеринбург"**
3. Пишет возражение: **"Кандидат говорит, что маленькая зарплата"**
4. Бот анализирует проект и генерирует ответ:
   - Анализ возражения
   - Конкретные аргументы из проекта
   - Готовый текст для разговора

## 🔒 Безопасность

⚠️ **Внимание:** API ключ сейчас встроен в код. Для продакшена:

1. Удалите ключ из `app.py`
2. Используйте переменные окружения на хостинге:
   ```python
   import os
   GEMINI_API_KEY = os.getenv("GEMINI_API_KEY")
   ```
3. Добавьте ключ в настройки Hugging Face Spaces / Render

## 📞 Если что-то не работает

1. Проверьте API ключ: https://aistudio.google.com/app/apikey
2. Убедитесь, что все зависимости установлены: `pip install -r requirements.txt`
3. Посмотрите логи приложения в файле `server.log`
