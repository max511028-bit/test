# 🤖 Помощник рекрутера для отработки возражений

Веб-приложение на базе ИИ для помощи рекрутерам в отработке возражений кандидатов.

## 🚀 Возможности

- **Интеграция с Google Gemini** - использует бесплатную модель GPT-4o уровня
- **Работа с проектами** - загрузка данных из Google Таблиц
- **База знаний** - использование накопленных скриптов и лучших практик
- **Умные ответы** - генерация персонализированных ответов на возражения
- **Веб-интерфейс** - удобный интерфейс для рекрутеров

## 📋 Требования

- Python 3.8+
- Аккаунт Google с API ключом Gemini
- Доступ к Google Таблицам (опционально)

## 🔧 Установка

### 1. Клонирование репозитория

```bash
git clone <ваш-репозиторий>
cd recruiter-bot
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Настройка API ключа

API ключ уже встроен в код (`app.py`). Если нужно заменить:

```python
GEMINI_API_KEY = "ваш-новый-ключ"
```

## 🎯 Запуск приложения

```bash
streamlit run app.py
```

Приложение откроется в браузере по адресу `http://localhost:8501`

## 📖 Как использовать

1. **Введите название проекта** - например, "Впрок Екатеринбург"
2. **Опишите возражение** - например, "Кандидат говорит, что маленькая зарплата"
3. **Получите рекомендацию** - ИИ сгенерирует ответ с учетом данных проекта

## 🔗 Интеграция с Google Таблицами

Для подключения к вашим Google Таблицам:

1. Создайте сервисный аккаунт в Google Cloud Console
2. Скачайте JSON-файл с ключами
3. Предоставьте доступ сервисному аккаунту к вашим таблицам
4. Обновите функцию `load_projects_data()` в `app.py`

Пример кода для подключения:

```python
from google.oauth2.service_account import Credentials
import gspread

def load_projects_data():
    scope = ["https://spreadsheets.google.com/feeds"]
    creds = Credentials.from_service_account_file(
        'path/to/credentials.json', scopes=scope
    )
    client = gspread.authorize(creds)
    sheet = client.open("Название таблицы").sheet1
    data = sheet.get_all_records()
    return data
```

## 🌐 Развертывание

### Вариант 1: Hugging Face Spaces (бесплатно)

1. Создайте аккаунт на [Hugging Face](https://huggingface.co/)
2. Создайте новый Space с типом "Streamlit"
3. Загрузите файлы проекта
4. Добавьте secrets для API ключей

### Вариант 2: Render (бесплатно)

1. Создайте аккаунт на [Render](https://render.com/)
2. Создайте новый Web Service
3. Подключите репозиторий GitHub
4. Укажите команду запуска: `streamlit run app.py --server.port $PORT`

### Вариант 3: Локальный сервер

Для использования внутри компании запустите на локальном сервере:

```bash
streamlit run app.py --server.address 0.0.0.0 --server.port 8501
```
